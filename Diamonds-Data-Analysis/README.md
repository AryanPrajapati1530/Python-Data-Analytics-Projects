# 💎Diamonds Dataset Analysis Using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Diamonds dataset using Python. The objective is to understand the factors affecting diamond prices and identify meaningful business insights through statistical analysis and data visualization.

## Dataset Information

The dataset contains information about diamond characteristics such as:

* Carat
* Cut
* Color
* Clarity
* Depth
* Table
* Price
* Physical dimensions (x, y, z)

Total Records: 53,940

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

### Data Exploration

* Dataset Overview

### Exploratory Data Analysis

* Price Distribution Analysis

* Color Distribution Analysis
* Clarity Distribution Analysis

### Feature Engineering

* Price Per Carat Calculation
* Category Encoding

### Additional Business Insights

* Most valuable diamond categories
* Best value-for-money diamonds
* Relationship between diamond characteristics and price
* Correlation analysis

## Key Findings

* Carat has the strongest influence on diamond price.
* Premium quality diamonds command higher prices.
* Color and clarity significantly impact valuation.
* Certain diamond categories provide better value per carat.
* Physical dimensions are strongly correlated with price.

## Conclusion

This analysis demonstrates how Exploratory Data Analysis can uncover valuable business insights and support data-driven decision-making in the gemstone industry.
